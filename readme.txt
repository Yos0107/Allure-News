== Description ==

Allure News is a free WordPress theme perfect for creating magazine-style websites. It's super easy to use, lightweight, and you can customize it effortlessly. The theme comes with a handy one-click demo import feature, 9 custom widgets, and lots of color choices. Plus, it's ready for ads and has SEO-friendly code. You can use it with popular plugins and it has a great layout for blogs, homepage ads, and related posts. It also offers various customization options like meta settings, header/footer choices, and different sidebars, including individual ones for single posts. Allure News is ready for Gutenberg, works with RTL languages, and is translation-ready. It's a user-friendly and versatile theme for your WordPress site!. View Demos: https://allure.candidthemes.com/allure-news

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.
